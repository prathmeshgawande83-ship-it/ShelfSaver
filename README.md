# 🌿 ShelfSaver

Save Money. Save Food. Reduce Waste.

ShelfSaver is a near-expiry product marketplace that helps customers buy discounted products while reducing waste.

## 🚀 Live Demo
https://shelf-saver-two.vercel.app

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript
- Vercel Deployment

## ✨ Features
- Responsive E-Commerce UI
- Product Listings
- Product Categories
- Shopping Cart
- Seller Dashboard
- Login/Signup UI
- Dark Mode
- Search & Filters

## Future Scope
- MERN Stack Backend
- MongoDB Database
- JWT Authentication
- Payment Gateway
